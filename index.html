Добро пожаловать в Myata

Полная документация по всем модулям и командам бота.

💡
Все слэш-команды начинаются с /. Некоторые команды требуют особых прав — смотри значок рядом с командой. Зелёный — для всех, жёлтый — модераторам, красный — только администраторам.
Значки прав
Все — Доступно всем участникам
Мод — Только для модераторов
Адм — Только для администраторов
